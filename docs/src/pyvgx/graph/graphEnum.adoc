[[relationships]]
== Relationships

[[graphenumrelationship]]
=== pyvgx.Graph.EnumRelationship()

[source, python]
----
pyvgx.Graph.EnumRelationship( relationship_name )
----

Return enumeration code for _relationship_name_. New enumeration mapping is created if _relationship_name_ is previously undefined.

[[graphrelationship]]
=== pyvgx.Graph.Relationship()

[source, python]
----
pyvgx.Graph.Relationship( code )
----

Return the relationship name associated with enumeration _code_ if it exists, otherwise raise <<../reference.adoc#EnumerationError_exc, `pyvgx.EnumerationError`>>.

[[graphrelationships]]
=== pyvgx.Graph.Relationships()

[source, python]
----
pyvgx.Graph.Relationships()
----

Return a dictionary of all existing { _relationship_name_ : _code_ } mappings.
