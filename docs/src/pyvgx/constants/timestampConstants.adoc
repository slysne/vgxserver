These constants represent valid internal timestamp limits.

|===
|Constant |Integer Value |Time (GMT)

|<<T_MIN>>
| 1
| Thu Jan 01 00:00:01 1970

|<<T_MAX>>
| 4,102,444,799
| Thu Dec 31 23:59:59 2099

|<<T_NEVER>>
| 4,102,444,800
| Infinity

|===

[[T_MIN]]
== pyvgx.T_MIN

`pyvgx.T_MIN = 1`

Minimum timestamp, in seconds in 1970.

[[T_MAX]]
== pyvgx.T_MAX

`pyvgx.T_MAX = 4102444799`

Maximum timestamp, in seconds in 1970.

[[T_NEVER]]
== pyvgx.T_NEVER

`pyvgx.T_NEVER = 4102444800`

Infinitely far into the future.
