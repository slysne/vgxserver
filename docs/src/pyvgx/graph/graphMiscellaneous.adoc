[[graphshowvertex]]
== pyvgx.Graph.ShowVertex()

Print vertex details

[source, python]
----
pyvgx.Graph.ShowVertex( id[, timeout ] )
----

[[graphvertexdescriptor]]
== pyvgx.Graph.VertexDescriptor()

Return vertex descriptor

[source, python]
----
pyvgx.Graph.VertexDescriptor( id )
----

[[graphshowopenvertices]]
== pyvgx.Graph.ShowOpenVertices()

Print open vertices

[source, python]
----
pyvgx.Graph.ShowOpenVertices()
----

[[graphdebugprintallocators]]
== pyvgx.Graph.DebugPrintAllocators()

Print internal allocator information to stdout. Specify allocator _name_ to dump a specific allocator.

[source, python]
----
pyvgx.Graph.DebugPrintAllocators( [ name ] )
----

[[graphdebugcheckallocators]]
== pyvgx.Graph.DebugCheckAllocators()

Run internal allocator diagnostics. Specify _name_ to run diagnostics for a specific allocator.

[source, python]
----
pyvgx.Graph.DebugCheckAllocators( [ name ] )
----

[[graphdebuggetobjectbyaddress]]
== pyvgx.Graph.DebugGetObjectByAddress()

Return a VGX object located as memory _address_.

[source, python]
----
pyvgx.Graph.DebugGetObjectByAddress( address )
----

[[graphdebugfindobjectbyidentifier]]
== pyvgx.Graph.DebugFindObjectByIdentifier()

Return a VGX object with a specific _identifier_.

[source, python]
----
pyvgx.Graph.DebugFindObjectByIdentifier( identifier )
----

[[graphdebugdumpgraph]]
== pyvgx.Graph.DebugDumpGraph()

Print internal graph information.

[source, python]
----
pyvgx.Graph.DebugDumpGraph()
----
